## Simple Interest Calculator

This repository contains a simple interest calculator script written in Bash.

## Description

The script calculates simple interest given the following inputs from the user:

- **p** – Principal amount
- **t** – Time period in years
- **r** – Annual rate of interest

## Formula

simple interest = p * t * r / 100

## Usage

Run the script in your terminal:

```bash
bash simple-interest.sh
```

You will be prompted to enter:
1. The principal amount
2. The time period in years
3. The rate of interest per year

The script will then output the calculated simple interest.
